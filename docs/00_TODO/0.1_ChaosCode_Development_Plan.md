# ChaosCode 开发计划

## 项目概述

**ChaosCode** 是一个基于 Python 的 AI 编程助手 CLI 工具，借鉴 opencode、gemini-cli、ms-agent 三个优秀开源项目的精华设计。

### 项目定位

- **语言**: Python 3.10+
- **界面**: TUI (Terminal User Interface)
- **架构**: 模块化、可扩展
- **核心功能**: 代码编辑、文件操作、终端执行、LLM 集成、MCP 协议支持

## MVP 详细设计

### 工具基类设计

```python
from abc import ABC, abstractmethod
from typing import Any, Dict, Optional
from pydantic import BaseModel

class ToolResult(BaseModel):
    """工具执行结果 [参考: MS-Agent]"""
    success: bool
    output: str
    error: Optional[str] = None
    metadata: Dict[str, Any] = {}

class ToolBase(ABC):
    """工具基类 [参考: OpenCode 整体架构 + Gemini CLI 确认机制]"""

    name: str                    # [OpenCode] 工具名称
    description: str             # [OpenCode] 工具描述
    parameters_schema: Dict[str, Any]  # [OpenCode] JSON Schema

    @abstractmethod
    async def execute(self, params: Dict[str, Any], context: 'ToolContext') -> ToolResult:
        """执行工具 [OpenCode + MS-Agent]"""
        pass

    def should_confirm(self, params: Dict[str, Any]) -> bool:
        """是否需要用户确认 [Gemini CLI]"""
        return False

    def get_description(self, params: Dict[str, Any]) -> str:
        """获取本次调用的描述 [Gemini CLI]"""
        return f"执行 {self.name}"
```

### Agent 设计

```python
from typing import List, AsyncGenerator
from .llm import LLM, Message
from .tools import ToolRegistry

class Agent:
    """Coding Agent [参考: OpenCode Agent 架构 + MS-Agent LLMAgent]"""

    def __init__(
        self,
        llm: LLM,                  # [MS-Agent] LLM 抽象
        tools: ToolRegistry,       # [OpenCode] 工具注册表
        max_turns: int = 20
    ):
        self.llm = llm
        self.tools = tools
        self.max_turns = max_turns
        self.messages: List[Message] = []

    async def run(self, user_input: str) -> AsyncGenerator[Message, None]:
        """运行 Agent 主循环 [OpenCode 工具调用循环]"""
        self.messages.append(Message(role="user", content=user_input))

        for _ in range(self.max_turns):
            # 调用 LLM [MS-Agent]
            response = await self.llm.generate(
                messages=self.messages,
                tools=self.tools.get_schemas()
            )

            self.messages.append(response.message)
            yield response.message

            # 检查是否需要工具调用 [OpenCode]
            if response.tool_calls:
                for tool_call in response.tool_calls:
                    result = await self.execute_tool(tool_call)
                    self.messages.append(result)
                    yield result
            else:
                # 没有工具调用，结束循环
                break

    async def execute_tool(self, tool_call: ToolCall) -> Message:
        """执行工具调用 [OpenCode + Gemini CLI 确认机制]"""
        tool = self.tools.get(tool_call.name)
        if not tool:
            return Message(
                role="tool",
                content=f"错误：未知工具 {tool_call.name}",
                tool_call_id=tool_call.id
            )

        # 检查是否需要确认 [Gemini CLI]
        if tool.should_confirm(tool_call.arguments):
            confirmed = await self.request_confirmation(tool, tool_call)
            if not confirmed:
                return Message(
                    role="tool",
                    content="用户取消了操作",
                    tool_call_id=tool_call.id
                )

        # 执行工具 [OpenCode]
        result = await tool.execute(tool_call.arguments, self.context)

        return Message(
            role="tool",
            content=result.output,
            tool_call_id=tool_call.id
        )
```
