# ChaosCode 开发计划

## 项目概述

**ChaosCode** 是一个基于 Python 的 AI 编程助手 CLI 工具，借鉴 opencode、gemini-cli、ms-agent 三个优秀开源项目的精华设计。

### 项目定位

- **语言**: Python 3.10+
- **界面**: TUI (Terminal User Interface)
- **架构**: 模块化、可扩展
- **核心功能**: 代码编辑、文件操作、终端执行、LLM 集成、MCP 协议支持

## 参考项目精华提炼

### 1. OpenCode (TypeScript/Bun)

**技术栈**: TypeScript + Bun + AI SDK

**核心架构特点**:
- Client/Server 架构，支持远程控制
- 内置 LSP 支持，代码智能补全
- Agent 系统：build（全功能）和 plan（只读规划）两种模式
- 权限系统：细粒度的工具调用权限控制
- Skill 系统：可扩展的技能模块

**工具设计**:

### 2. Gemini CLI (TypeScript)

**技术栈**: TypeScript + Zod 验证

**核心架构特点**:
- 模块化的工具系统
- Subagent 架构支持复杂任务分解
- 确认机制（Confirmation Bus）处理危险操作
- 丰富的测试覆盖

**工具设计**:
```typescript
interface ToolInvocation<TParams, TResult> {
  params: TParams;
  getDescription(): string;
  toolLocations(): ToolLocation[];
  shouldConfirmExecute(signal: AbortSignal): Promise<ToolCallConfirmationDetails | false>;
  execute(signal: AbortSignal): Promise<TResult>;
}
```

**借鉴要点**:
- 工具调用的确认机制
- Zod schema 验证模式
- Subagent 任务委派机制

### 3. MS-Agent (Python)

**技术栈**: Python + OmegaConf + MCP

**核心架构特点**:
- LLMAgent 核心类，支持工具调用
- MCP (Model Context Protocol) 完整支持
- 多种 LLM 后端（OpenAI、Anthropic、DeepSeek、ModelScope）
- Memory 系统（长期/短期记忆）
- Skills 系统（Anthropic Agent Skills 协议实现）

**Agent 设计**:

**借鉴要点**:
- Python 原生实现，直接可用
- MCP 协议集成方式

## MVP 详细设计

### 工具基类设计

```python
from abc import ABC, abstractmethod
from typing import Any, Dict, Optional
from pydantic import BaseModel

class ToolResult(BaseModel):
    """工具执行结果"""
    success: bool
    output: str
    error: Optional[str] = None
    metadata: Dict[str, Any] = {}

class ToolBase(ABC):
    """工具基类"""

    name: str  # 工具名称
    description: str  # 工具描述
    parameters_schema: Dict[str, Any]  # JSON Schema

    @abstractmethod
    async def execute(self, params: Dict[str, Any], context: 'ToolContext') -> ToolResult:
        """执行工具"""
        pass

    def should_confirm(self, params: Dict[str, Any]) -> bool:
        """是否需要用户确认"""
        return False

    def get_description(self, params: Dict[str, Any]) -> str:
        """获取本次调用的描述"""
        return f"执行 {self.name}"
```

### Agent 设计

```python
from typing import List, AsyncGenerator
from .llm import LLM, Message
from .tools import ToolRegistry

class Agent:
    """Coding Agent"""

    def __init__(
        self,
        llm: LLM,
        tools: ToolRegistry,
        max_turns: int = 20
    ):
        self.llm = llm
        self.tools = tools
        self.max_turns = max_turns
        self.messages: List[Message] = []

    async def run(self, user_input: str) -> AsyncGenerator[Message, None]:
        """运行 Agent 主循环"""
        self.messages.append(Message(role="user", content=user_input))

        for _ in range(self.max_turns):
            # 调用 LLM
            response = await self.llm.generate(
                messages=self.messages,
                tools=self.tools.get_schemas()
            )

            self.messages.append(response.message)
            yield response.message

            # 检查是否需要工具调用
            if response.tool_calls:
                for tool_call in response.tool_calls:
                    result = await self.execute_tool(tool_call)
                    self.messages.append(result)
                    yield result
            else:
                # 没有工具调用，结束循环
                break

    async def execute_tool(self, tool_call: ToolCall) -> Message:
        """执行工具调用"""
        tool = self.tools.get(tool_call.name)
        if not tool:
            return Message(
                role="tool",
                content=f"错误：未知工具 {tool_call.name}",
                tool_call_id=tool_call.id
            )

        # 检查是否需要确认
        if tool.should_confirm(tool_call.arguments):
            confirmed = await self.request_confirmation(tool, tool_call)
            if not confirmed:
                return Message(
                    role="tool",
                    content="用户取消了操作",
                    tool_call_id=tool_call.id
                )

        # 执行工具
        result = await tool.execute(tool_call.arguments, self.context)

        return Message(
            role="tool",
            content=result.output,
            tool_call_id=tool_call.id
        )
```
